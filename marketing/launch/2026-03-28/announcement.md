Hey,

I built gstack for engineering. Same idea, different team: mstack gives Claude Code a virtual marketing team.

One command — `/mlaunch` — reads your README and CHANGELOG and writes four things: landing page copy, an email announcement, social posts, and ad copy. Under 60 seconds. No SaaS tab, no account, no monthly fee.

The rest of the suite covers the full lifecycle. `/mbrand` infers your brand voice and writes a `BRAND.md` that every other skill uses. `/mcmo` produces a one-page positioning brief. `/memail` writes onboarding sequences. `/maudit` tells you what's broken in your marketing before you ship more of it.

Nine skills total. All open source. MIT licensed.

Install:

```
git clone --single-branch --depth 1 https://github.com/deadjim/mstack.git \
  ~/.claude/skills/mstack && cd ~/.claude/skills/mstack && ./setup
```

Then go to any repo and run `/mlaunch`.
